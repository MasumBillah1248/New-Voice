# New-Voice

Hi Masum

New Voice assistant is a python-based voice assistant that can interact with user through speech playing music randomly. telling jokes and having small talk.

This project uses speech recognition and text-to-speech (TTS) to provide a hands-free assistant experience similar to Iron Man's.



# Features

- Greet the user according to the time of day (morning , afternoon, evening)

- Recognize voice commands using Google Speech Recognition

- Speak respones using pyttsx3

- Time & Date announcements

- Wikipedia search with spoken summary

- Open websites like Google, Facebook, YouTube

- Play random music from a specified folder

- Open system applications: Calculator, Notepad, CMD

- Open Calendar (Google Calendar via browser)

- Tell jodes and respond to basic small talk

- Exit gracefully with a voice command



 

## Requirements

- Python 3.11 or higher


## How to run?

1. Create a virtual environment:

    ```bash
    conda create -n new_voice python=3.11 -y
    ```

2. Activate virtual environment:

    ```bash
    conda activate new_voice
    ```

3. Install required packages:

    ```bash
    pip install -r requirements.txt
    ```


4. Run the New_voice scripot:

    ```bash
    python new_voice.py
    ```
